# Nexacon SDK Example

This example demonstrates how to use the Nexacon Flutter SDK for P2P calling.

## Running the Example

1. Navigate to the example directory:
   ```bash
   cd example
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

## Features

- Initialize NexaconClient with API credentials
- Generate NX token for signaling
- Create CallManager for P2P calling
- Initiate and end calls
- Monitor call state changes
- Handle incoming calls

## Configuration

Before running, update the API credentials in the app:
- API Key
- Secret Key
- Your username
- Recipient's username
